# Функции в JavaScript. Пользовательские функции

## Что такое функции

Функция - это блок кода, который можно вызывать много раз.

функции нужны для: 
- повторного использования кода 

- упрощения программы 
- разделения логики

## Зачем нужны функции

Без функции:

``` js
console.log("Привет");
console.log("Привет");
console.log("Привет");
```

С функцией:

``` js
function sayHello() {
  console.log("Привет");
}

sayHello();
sayHello();
sayHello();
```

## Создание функции

``` js
function имя() {
  // код
}
```

Пример:

``` js
function hello() {
  console.log("Hello");
}
```

## Вызов функции

Функция выполняется только после вызова.

``` js
hello();
```

## Параметры функции

Параметры позволяют передавать данные внутрь функции.

``` js
function hello(name) {
  console.log("Привет, " + name);
}
```

## Несколько параметров

``` js
function sum(a, b) {
  console.log(a + b);
}
```

``` js
sum(5, 3);
```

## Возврат значения (`return`)

Функция может возвращать результат.

``` js
function sum(a, b) {
  return a + b;
}
```

## Пример `return`

``` js
let result = sum(2, 3);

console.log(result);
```

## Локальные переменные

``` js
function test() {
  let x = 10;
}
```

* переменная доступна только внутри функции

## Стрелочные функции

``` js
const hello = () => {
  console.log("Hello");
};
```
# Задание

1. Напишите функцию, возводящую целое число в квадрат.

    Ввод содержит целое число `N`.

    Выведите одно целое число — величину `N^2`.

2. Напишите функцию, возвращающую количество дней в месяце с указанным номером (нумерация начинается с единицы); год является невисокосным.

    Если передан некорректный номер месяца, требуется вернуть `0`.

    Ввод содержит целое число `M` — номер месяца.

    Выведите одно целое число — количество дней в заданном месяце невисокосного года.

3. Напишите функцию, возвращающую меньшее из двух чисел `A` и `B`. Если числа равны, нужно вывести любое из них.

    Ввод содержит целые числа `A` и `B`.

    Выведите одно целое число — минимум из `A` и `B`.
