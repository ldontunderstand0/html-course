## Что такое CSS

CSS (Cascading Style Sheets) - это язык стилей, который используется для оформления HTML-документов.

Проще: 
* HTML - структура
* CSS - внешний вид

С помощью CSS можно:
* менять цвета 
* задавать размеры 
* управлять расположением элементов 
* делать адаптивный дизайн

## Как подключить CSS к HTML

### 1. Внешний файл (лучший способ)

``` html
<link rel="stylesheet" href="styles.css">
```

* используется чаще всего
* удобно для больших проектов

### 2. Внутри документа

``` html
<style>
  body {
    background: lightgray;
  }
</style>
```

* используется редко (плохая практика)

### 3. Встроенный в элемент стиль

``` html
<p style="color: red;">Текст</p>
```

* используется редко (плохая практика)

## Синтаксис CSS

### Общая структура:

``` css
селектор {
  свойство: значение;
}
```

### Пример:

``` css
p {
  color: blue;
  font-size: 16px;
}
```

### Разбор:

- `p` - селектор
- `color` - свойство
- `blue` - значение

## Основные селекторы

``` css
p {} /* селектор по элементу - применится ко всем таким элементам */
.class {} /* селектор по классу - применится ко всем элементам класса */
#id {} /* селектор по id - применится к элементу с указанным id */
* {} /* селектор * - применится ко всем элементам на странице */
```

## Синтаксические правила CSS3

1.  Каждое свойство заканчивается `;`

2.  Используются `{ }`
3.  Разделитель `:`
4.  Комментарии:

``` css
/* комментарий */
```

## Обработка ошибок

Если ошибка:

``` css
p {
  color: bluue;
}
```

* свойство игнорируется, остальное работает

## Типы значений

### Цвета

``` css
color: red; /* именованный цвет */
color: #ff0000; /* HEX (шестнадцатеричный) формат */
color: rgb(255,0,0); /* rgb формат */
```

### Размеры

``` css
width: 100px;
font-size: 16px;
```

Единицы: 
* px - пиксели, это абсолютная единица, фиксированный размер, не зависит от других элементов, лучший вариант для мелких деталей

* % - — проценты, относительно родителя, 50% = половина ширины родительского элемента, лучший вариант для ширины
* em - % от родителя
* rem - % от корня, лучший вариант для текста

### Шрифты

``` css
font-family: "Arial";
```
