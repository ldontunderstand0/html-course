# Блоковая структура документа. Понятие контейнера

HTML-документ состоит из блоков (элементов), которые располагаются друг под другом.

Каждый блок: 
- занимает всю доступную ширину 
- начинается с новой строки

Примеры блочных элементов:

``` html
<div></div>
<p></p>
<h1></h1>
<section></section>
```

## Понятие контейнера

Контейнер - это элемент, который содержит внутри другие элементы.

``` html
<div class="container">
  <p>Текст внутри контейнера</p>
</div>
```

- `div` - контейнер
- `p` - вложенный элемент

## Нормальный поток документа

По умолчанию элементы располагаются сверху вниз.

``` html
<p>Первый</p>
<p>Второй</p>
<p>Третий</p>
```

## Свойство position

Определяет способ позиционирования элемента.

``` css
div {
  position: static;
}
```

### Возможные значения перемещения
``` css
top: 10px;
left: 20px;
right: 20px;
bottom: 10px;
```

## Виды позиционирования

### 1. static (по умолчанию)

- Элемент находится в нормальном потоке
- игнорирует top, left, right, bottom

``` css
div {
  position: static;
}
```

### 2. relative (относительное)

* остаётся в нормальном потоке
* занимает своё место как раньше
* но визуально сдвигается

``` css
div {
  position: relative;
  top: 10px;
  left: 20px;
}
```

### 3. absolute (абсолютное)

- другие элементы его не видят
- он может накладываться на них
- позиционируется относительно:

``` css
div {
  position: absolute;
  top: 0;
  right: 0;
}
```

- выходит из потока
- позиционируется относительно родителя

### Пример:

``` html
<div class="parent">
  <div class="child">Блок</div>
</div>
```

``` css
.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 0;
  right: 0;
}
```

## Свойство float

Позволяет обтекать элемент текстом

``` css
img {
  float: left;
}
```

``` html
<img src="img.jpg">
<p>Текст обтекает изображение</p>
```

## Свойство z-index

Определяет порядок наложения элементов

``` css
.box1 {
  position: absolute;
  z-index: 1;
}

.box2 {
  position: absolute;
  z-index: 2;
}
```

- большее значение - выше слой
