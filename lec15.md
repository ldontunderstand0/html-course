# Знакомство с JavaScript

## Что такое JavaScript

JavaScript - это язык программирования, который используется для
создания интерактивности на веб-страницах.

С его помощью можно: 
- реагировать на действия пользователя 

- изменять
HTML и CSS - работать с данными

## Особенности JavaScript

-   выполняется в браузере

-   работает после загрузки страницы
-   чувствителен к регистру
-   инструкции обычно заканчиваются `;`

## Подключение JavaScript

### 1. Внутри HTML

``` html
<script>
  alert("Привет");
</script>
```

### 2. Внешний файл

``` html
<script src="script.js"></script>
```

## Переменные

Переменные используются для хранения данных.

``` js
let x = 10;
const y = 20;
var z = 30;
```

Пояснение: 
- `let` - можно изменять 

- `const` - константа, изменять нельзя 
- `var` - устаревший способ

## Типы данных

``` js
let number = 10;
let text = "Hello";
let flag = true;
```

Основные типы: 
- Number 

- String 

- Boolean

## Операции

### Арифметические

``` js
let a = 5 + 3;
let b = 10 - 2;
let c = 4 * 2;
let d = 8 / 2;
```

### Сравнение

``` js
5 == "5";
5 === "5";
```

Пояснение: 
- `==` сравнивает значение 
- `===` сравнивает значение и тип

### Логические

``` js
true && false;
true || false;
!true;
```
Пояsнение: 
- `&&` - логическое И 

- `||` - логическое ИЛИ 
- `!` - логическое НЕ

## Вывод данных

``` js
alert("Привет");
console.log("Текст");
```
