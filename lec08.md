## Селекторы, псевдоэлементы и псевдоклассы

## Что такое селектор

Селектор - это часть CSS, которая указывает, к каким HTML-элементам
применять стили.

``` css
p {
  color: blue;
}
```

## Простые селекторы

### 1. Селектор по тегу

``` css
p {
  color: red;
}
```

### 2. Универсальный селектор

``` css
* {
  margin: 0;
  padding: 0;
}
```

* применяется ко всем элементам

### 3. Селектор класса

``` css
.box {
  background: gray;
}
```

HTML:

``` html
<div class="box"></div>
```

### 4. Селектор ID

``` css
#header {
  background: black;
}
```

HTML:

``` html
<div id="header"></div>
```

## Селекторы контекста (вложенности)

### Потомок

``` css
div p {
  color: blue;
}
```

* все p внутри div

HTML:

```html
<div>
  <p>Этот текст будет синим</p>
  <section>
    <p>Этот тоже будет синим</p>
  </section>
</div>

<p>Этот НЕ будет синим</p>
```

### Дочерний селектор

``` css
div > p {
  color: red;
}
```

* только прямые потомки

HTML:

```html
<div>
  <p>Этот текст будет красным</p>

  <section>
    <p>Этот НЕ будет красным</p>
  </section>
</div>
```

### Соседний селектор

``` css
h1 + p {
  color: green;
}
```

* первый p после h1

HTML:

```html
<h1>Заголовок</h1>
<p>Этот будет зелёным</p>

<p>Этот НЕ будет зелёным</p>
```

### Общий сосед

``` css
h1 ~ p {
  color: purple;
}
```

* все p после h1

HTML:

```html
<h1>Заголовок</h1>

<p>Этот будет фиолетовым</p>
<p>Этот тоже будет фиолетовым</p>

<div>
  <p>Этот НЕ будет (внутри div)</p>
</div>
```

## Псевдоклассы

Псевдоклассы — это специальные ключевые слова в CSS, которые описывают состояние элемента или его положение в документе.

Они добавляются к селектору через `:`

``` css
a:hover {
  color: red;
}
```

Псевдокласс применяется, когда элемент находится в определённом состоянии.

Например:

* пользователь навёл курсор
* кликнул
* ввёл текст в поле

### Примеры:

``` css
a:hover {} /* наведение мыши */ 
a:active {} /* нажатие */
a:visited {} /* посещённая ссылка */
input:focus {} /* фокус на элементе */ 
```

## Псевдоэлементы

Псевдоэлементы — это специальные конструкции в CSS, которые позволяют стилизовать часть элемента.

; записываются через `::`

``` css
p::first-letter { /* первая буква */
  font-size: 30px;
}
```

### Примеры:

``` css
p::first-line {} /* первая строка */ 
div::before {}  /* элемент перед */ 
div::after {} /* элемент после */ 
```
* `::before` и `::after` НЕ работают без `content`

### Пример before/after:

``` css
div::before {
  content: "★ ";
}
```
