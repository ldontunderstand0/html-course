# Циклы в JavaScript (for, while)

## Что такое циклы

Циклы позволяют выполнять один и тот же код несколько раз.

используются, когда нужно: 
- повторять действия 

- обрабатывать данные 
- работать со списками

## Зачем нужны циклы

Без цикла:

``` js
console.log(1);
console.log(2);
console.log(3);
```

С циклом:

``` js
for (let i = 1; i <= 3; i++) {
  console.log(i);
}
```

* код становится короче и удобнее

## Цикл for

Используется, когда известно количество повторений.

``` js
for (начало; условие; шаг) {
  // код
}
```

### Разбор структуры

``` js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

Пояснение: 
- `let i = 0` - старт

 - `i < 5` - условие 
 - `i++` - шаг

### Пример

``` js
for (let i = 1; i <= 5; i++) {
  console.log("Шаг: " + i);
}
```

## Цикл while

Используется, когда количество повторений заранее неизвестно.

``` js
while (условие) {
  // код
}
```

### Пример

``` js
let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}
```

Важно: 
- обязательно изменять переменную - иначе будет бесконечный цикл

## Отличие for и while

  | Цикл | Когда использовать |
  | - | - |
  for | известно количество
  while | неизвестно заранее
