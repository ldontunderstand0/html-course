# Визуальные функции в CSS3

В `CSS3` есть специальные визуальные функции, которые позволяют изменять внешний вид элементов: размывать, менять прозрачность, добавлять тени и эффекты.

Чаще всего они применяются через свойство:

```css
filter: значение;
```
## 1. `blur()` — размытие

Функция делает элемент размытым.

```css
filter: blur(5px);
```

* значение задаётся в пикселях чем больше значение — тем сильнее размытие

### Пример:

```css
img {
  filter: blur(3px);
}
```
* изображение становится размытым

## 2. `opacity` — прозрачность

Определяет степень прозрачности элемента.

```css
opacity: 0.5;
```

- `1` — полностью видимый
- `0` — полностью прозрачный

Важно:
применяется ко всему элементу (включая текст и вложенные элементы) в отличие от `filter` — это отдельное свойство

### Пример:

```css
div {
  opacity: 0.7;
}
```
* элемент становится полупрозрачным

## 3. `drop-shadow()` — тень

Добавляет тень к элементу (особенно хорошо работает с изображениями и `PNG`).

```css
filter: drop-shadow(5px 5px 10px black);
```
* `5px` — смещение по `X`
* `5px` — смещение по `Y`
* `10px` — размытие
* `black` — цвет

### Пример:
```css
img {
  filter: drop-shadow(2px 2px 5px gray);
}
```
* появляется мягкая тень

## 4. `grayscale()` — чёрно-белый эффект

Преобразует изображение в чёрно-белое.

```css
filter: grayscale(100%);
```
* 0% — без эффекта
* 100% — полностью ч/б

### Пример:

```css
img {
  filter: grayscale(100%);
}
```
* изображение становится чёрно-белым

## 5. `invert()` — инверсия цветов

Инвертирует цвета элемента.

```css
filter: invert(100%);
```

цвета становятся противоположными
* чёрный → белый
* белый → чёрный

### Пример:

```css
img {
  filter: invert(100%);
}
```
*  получается "негатив" изображения

## Комбинирование функций

Можно применять несколько эффектов одновременно:

```css
filter: blur(2px) grayscale(100%) invert(50%);
```
* эффекты применяются по порядку

### Важно
* `filter` работает визуально (не влияет на размеры блока)

* можно применять к изображениям, блокам, тексту:
    | Функция | К чему применяется |
    |-|-|
    | blur | всё (текст, блоки, img) |
    | opacity | всё |
    | drop-shadow | лучше для img / SVG |
    | grayscale | всё с цветом |
    | invert | всё |
* эффекты могут снижать производительность при большом количестве
