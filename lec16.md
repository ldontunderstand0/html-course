# Условия в JavaScript. Конструкция if-else

## Что такое условия

Условия позволяют программе принимать решения в зависимости от данных.

* программа проверяет выражение и выбирает, что выполнить

## Логическое выражение

Условие всегда возвращает:

-   true (истина)

-   false (ложь)

``` js
5 > 3 // true
10 == 5 // false
```

## Конструкция if

Позволяет выполнить код, если условие истинно.

``` js
if (условие) {
  // код
}
```

### Пример:

``` js
let age = 18;

if (age >= 18) {
  console.log("Доступ разрешён");
}
```

## Конструкция if-else

Используется, когда нужно выполнить один из двух вариантов.

``` js
if (условие) {
  // если true
} else {
  // если false
}
```

### Пример:

``` js
let age = 16;

if (age >= 18) {
  console.log("Совершеннолетний");
} else {
  console.log("Несовершеннолетний");
}
```

------------------------------------------------------------------------

## Конструкция else if

Позволяет проверять несколько условий.

``` js
if (условие1) {
} else if (условие2) {
} else {
}
```

### Пример:

``` js
let score = 75;

if (score >= 90) {
  console.log("Отлично");
} else if (score >= 70) {
  console.log("Хорошо");
} else {
  console.log("Плохо");
}
```

## Вложенные условия

``` js
let age = 20;
let hasTicket = true;

if (age >= 18) {
  if (hasTicket) {
    console.log("Вход разрешён");
  }
}
```

------------------------------------------------------------------------

## Операторы сравнения

``` js
>   // больше
<   // меньше
>=  // больше или равно
<=  // меньше или равно
==  // равенство по значению
=== // равенство по значению и типу
!=  // не равно
```

## Логические операторы

``` js
&& // логическое И
|| // логическое ИЛИ
!  // логическое НЕ
```
